# Project Overview

This project consists of a multi-page website developed for the CNIT131 course at City College of San Francisco. The site serves as a professional portfolio and blog to track academic progress in Computer Science and Cybersecurity. It features a homepage, a cybersecurity information page, a media gallery, and a technical reference page for networking protocols.

---

## File Structure

* **home_cg.html**: The landing page containing a professional summary, academic goals (AI Red-Team/Offensive Security), and a list of current projects such as OWASP Juice Shop and Bug Crowd Bounties.
* **p01.html**: An educational page defining offensive and defensive cybersecurity roles. It includes categorized resource links to platforms like GitHub, OWASP, and Digital Ocean.
* **p02.html**: A gallery page utilizing the fit-content property to display embedded YouTube videos, GIPHY animations, and industry-related imagery.
* **p03.html**: A technical reference page containing data tables for Subnet Masks and common Networking Ports/Protocols.

---

## Technical Features

### HTML and Layout
* Semantic HTML5 elements including headers, footers, and navigation lists.
* Responsive navigation menus positioned using absolute coordinates.
* Data presentation using HTML tables with fixed layouts for technical accuracy.

### CSS Styling
* Custom typography using Courier New and monospace fonts to maintain a technical aesthetic.
* Advanced container sizing using width: fit-content to wrap borders tightly around dynamic media.
* Viewport-based scaling (vh/vw) for consistent element positioning across different screen sizes.
* Background image integration with centered positioning and no-repeat rules.
